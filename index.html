MultiMC

  1. Download the right file. (DO NOT EXTRACT IT)
  2. In MultiMC, press Add Instance
  3. Click on Import from zip
  4. Click on Browse, then select the zip file you downloaded.
  5. Make sure the correct file is selected
  6. Click OK and it will magically install itself.

Once done, just press play and have fun!

Curseforge

  1. Download the right file. (DO NOT EXTRACT IT)
  2. In CurseForge, press Import

    3. If there is no Import button, press New, and then Import at the top in small text.

  3. Select the zip file you downloaded.

Once done, just press play and have fun!

I just put this option cuz I know there are lazy people around here. cough cough ghost

  1. Download the right file. Extract/open it.
  2. In the windows search bar, type %appdata%.
  3. Navigate to /.minecraft/.
  4. Inside the file you downloaded or extracted, navigate to /.minecraft/.
  5. Transfer all files from this folder to the folder you opened in step 3.
  6. If your Minecraft launcher is already open, close it.
  7. Download FabricMC's installer Here.
  8. Open the installer.
  9. Under Minecraft Version, select 1.21.1.
  10. Under Fabric Version, select 0.16.9.
  11. Keep create new profile on, and press install.
  12. Now, open your Minecraft launcher
  13. Under Installations, find the new installation created by Fabric, and then select Edit.
  14. Click More Options.
  15. Under JVM Arguments, change the number in -Xmx2G to either 4 or higher, depending on how much ram your computer has.
  16. Finally, click play and pray nothing exploded.
